# data/raw/

This directory holds all raw input data. **Contents are gitignored.**

See [`../README.md`](../README.md) for acquisition instructions, licenses,
and file-size information for each source.

Expected files after download:

```
data/raw/
├── hcad_parcels_raw.gpkg       (~93 MB)   HCAD parcel polygons
├── harvey_depth_brays.tif      (~153 MB)  FEMA Harvey flood depth raster
├── ghsl_2015.tif               (~43 MB)   GHSL ISA 2015
├── dem.tif                     (~24 MB)   3DEP 10m DEM
├── nlcd_2016_brays.tif         (~2 MB)    NLCD 2016 (clipped)
├── nlcd_2016_brays_26915.tif   (~1 MB)    NLCD 2016 (reprojected)
├── nhd_streams_brays.gpkg      (~0.1 MB)  NHD flowlines
└── osm_streets.gpkg            (~22 MB)   OSM street network
```
