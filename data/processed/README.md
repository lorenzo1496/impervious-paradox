# data/processed/

This directory holds intermediate files generated by the Stage 1 pipeline
scripts. **Contents are gitignored.**

Regenerate by running from the repo root:

```bash
python src/01_data/compute_hand_10m.py
python src/01_data/add_nlcd_filter.py
```

Expected files after running Stage 1:

```
data/processed/
├── feature_matrix.gpkg         (~47 MB)   Parcel features with geometry
├── feature_matrix.csv          (~27 MB)   Parcel features (tabular)
├── harvey_depth_reproj.tif     (~379 MB)  Harvey depth reprojected to EPSG:26915
└── hand_brays.tif              (~0.4 MB)  HAND raster (30m, legacy)
```

The `feature_matrix` files are the entry point for all downstream scripts
(`src/02_modeling/` onward).
